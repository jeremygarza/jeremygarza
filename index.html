    BandSpy

    This web app was a team effor in General Assembly. For this project we had 5 days to come up an idea and do as much as we can with that idea using Ruby on Rails. Before this we had three weeks of classes on Ruby using Rails, Senatra and PostgreSQL as our database. The app lets a user sign up for a secure account then lets the user search for thier favorite artists and save them. Then they can see all saved artists and see when they are playing near them and buy tickets. The app uses the Bandsintown API.
